# Go로 만드는 "Hello World!"

이 문서는 Go 언어로 작성된 가장 기본적인 "Hello World!" 프로그램에 대해 설명합니다. Go 언어를 처음 배우는 분들을 위해 코드의 각 부분이 무엇을 의미하는지 자세히 설명합니다.

## 코드

```go
// 이 파일은 'main' 패키지에 속합니다. Go 프로그램은 패키지로 구성되며, 'main' 패키지는 실행 가능한(실행할 수 있는) 프로그램을 만듭니다.
package main

// "fmt" 패키지를 가져옵니다. "fmt"는 "format"의 줄임말로,
// 입출력(예: 화면에 텍스트 출력) 기능을 제공하는 표준 라이브러리입니다.
import "fmt"

// 'main' 함수는 Go 프로그램의 시작점입니다.
// 프로그램이 실행될 때 가장 먼저 호출되는 함수입니다.
func main() {
	// 'fmt' 패키지의 'Println' 함수를 사용하여 "Hello World!" 문자열을 화면에 출력합니다.
	// 'Println'은 괄호 안의 내용을 출력하고 새 줄(개행)을 추가합니다.
	fmt.Println("Hello World!")
}
```

## 코드 설명

### `package main`

*   **설명**: 모든 Go 프로그램은 `package` 선언으로 시작합니다. `package main`은 이 파일이 독립적으로 실행될 수 있는 실행 가능한 프로그램(executable program)의 일부임을 나타냅니다. Go 애플리케이션의 시작점은 항상 `main` 패키지에 있습니다.

### `import "fmt"`

*   **설명**: `import` 키워드는 외부 패키지를 현재 파일로 가져올 때 사용됩니다. 여기서 `"fmt"`는 "format"의 약자로, 기본적인 입출력(예: 텍스트 출력, 입력 받기) 기능을 제공하는 표준 라이브러리 패키지입니다. `fmt.Println` 함수와 같이 `fmt` 패키지 안에 있는 함수를 사용하기 위해 이 패키지를 가져와야 합니다.

### `func main() { ... }`

*   **설명**: `func` 키워드는 함수를 선언할 때 사용됩니다. `main()` 함수는 Go 프로그램의 특별한 함수로, 프로그램이 실행될 때 가장 먼저 호출되는 진입점(entry point)입니다. 모든 실행 가능한 Go 프로그램은 반드시 하나의 `main` 함수를 가지고 있어야 합니다. 이 함수 안에 작성된 코드가 프로그램이 시작될 때 실행됩니다.

### `fmt.Println("Hello World!")`

*   **설명**: 이 줄은 실제로 "Hello World!" 텍스트를 화면에 출력하는 역할을 합니다.
    *   `fmt`: 위에서 가져온 `fmt` 패키지를 의미합니다.
    *   `.`: 패키지 내의 함수나 변수에 접근할 때 사용합니다.
    *   `Println`: `fmt` 패키지 안에 정의된 함수 중 하나로, 괄호 `()` 안의 내용을 콘솔에 출력하고 자동으로 줄바꿈(새 줄)을 해줍니다. (`Print` 함수는 줄바꿈을 하지 않습니다.)
    *   `"Hello World!"`: `Println` 함수에 전달된 문자열(텍스트)입니다. 큰따옴표로 묶인 것은 컴퓨터에게 이것이 텍스트임을 알려줍니다.

## 프로그램 실행 방법

1.  **파일 저장**: 위 코드를 `main.go`라는 이름으로 저장합니다. (예: `cmd/helloworld/main.go`)
2.  **터미널 열기**: 터미널 또는 명령 프롬프트를 엽니다.
3.  **디렉토리 이동**: `main.go` 파일이 있는 디렉토리로 이동합니다.
    ```bash
    cd Core/TuckerGolang/cmd/helloworld
    ```
4.  **실행**: 다음 명령어를 입력하여 프로그램을 실행합니다.
    ```bash
    go run main.go
    ```
5.  **결과 확인**: 터미널에 다음과 같은 출력이 나타날 것입니다.
    ```
    Hello World!
    ```

이것으로 Go 언어의 첫 "Hello World!" 프로그램에 대한 설명을 마칩니다.
