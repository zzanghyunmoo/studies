# N8N 소개

## 왜 N8N인가?

최고의 코드는 존재하지 않는 코드라는 말이 있습니다. 코드가 없다면 관련된 이슈, 버그, 그리고 유지보수 부담도 사라지기 때문입니다. n8n은 이러한 철학을 바탕으로 코드 한 줄 없이도 다양한 업무를 자동화할 수 있도록 돕는 도구입니다.

구글 독스, 슬랙, 디스코드 등 수많은 서비스 및 도구들과 손쉽게 통합할 수 있으며, 드래그 앤 드롭 방식만으로도 방대한 자동화 작업을 구현할 수 있습니다.

또한, n8n의 큰 장점은 단순히 '노코드(No-code)' 플랫폼에 머무르지 않는다는 점입니다. 필요에 따라 적재적소에 코드를 작성하여 워크플로우의 기능을 확장하고 복잡한 로직을 구현할 수 있는 '로우코드(Low-code)'의 유연성을 제공합니다.

당신이 n8n을 배워야 하는 세 가지 주요 이유:

1.  **오픈소스 및 무료**: n8n은 오픈소스 프로젝트이므로 누구나 자유롭게 사용하고 기여할 수 있으며, 기본 기능을 무료로 활용할 수 있습니다.
2.  **쉬운 학습 곡선**: 직관적인 UI와 드래그 앤 드롭 방식으로 인해 자동화 워크플로우를 쉽게 배우고 구축할 수 있습니다.
3.  **코드 확장성**: 노코드의 편리함과 더불어, 필요시 코드를 직접 작성하여 무한한 가능성과 유연성을 확보할 수 있습니다.

## N8N은 무엇인가?

n8n은 복잡한 워크플로우를 시각적으로 구축하고 실행할 수 있도록 돕는 강력한 자동화 솔루션입니다. 여기서 워크플로우란 여러 단계의 작업을 미리 정의된 순서에 따라 자동으로 실행하는 일련의 과정을 의미합니다.

**예시**:
*   특정 키워드가 포함된 새로운 트윗이 올라왔을 때 자동으로 슬랙(Slack) 채널에 알림을 보내는 워크플로우.
*   새로운 이메일을 받았을 때 첨부 파일을 자동으로 구글 드라이브에 저장하고, 파일 정보를 구글 스프레드시트에 기록하는 워크플로우.
*   웹사이트에서 데이터를 스크래핑하여 주기적으로 보고서를 생성하고 이메일로 발송하는 워크플로우.

n8n은 웹훅(Webhook)을 포함하여 대부분의 온라인 서비스와 통합할 수 있는 유연성을 제공합니다. 다시 한번 강조하지만, n8n은 단순한 노코드 플랫폼이 아닙니다. 노코드 플랫폼은 때때로 명확한 한계에 부딪히기 쉽지만, n8n은 코드를 삽입할 수 있는 강력한 기능 덕분에 사실상 무한한 자동화 가능성을 가지게 됩니다.

## n8n 학습 가이드

이 문서는 n8n 학습을 위한 다음 구조를 제안합니다:

1.  n8n 클라우드를 이용한 기초 학습 및 워크플로우 생성
2.  다양한 실제 워크플로우를 만들면서 n8n 심화 학습
3.  오픈소스 n8n 버전 설치 및 운영 방법 (로컬 또는 서버 환경)
4.  복잡하고 실용적인 워크플로우 설계 및 구축

## 요구 사항

 n8n을 효과적으로 활용하기 위해 다음의 기초 지식이 있으면 좋습니다:

*   **Python 기초**: 특히 변수, 함수 등 프로그래밍의 기본적인 개념을 이해하고 있다면, n8n 내에서 코드를 활용할 때 큰 도움이 됩니다.
*   **인터넷 용어 이해**: API(Application Programming Interface), API Key, 웹훅(Webhook) 등 웹 서비스 연동에 필요한 기본적인 인터넷 및 개발 용어에 대한 이해가 필요합니다.

## 면책 조항

*   **무료 플랜 변동 가능성**: n8n의 무료 플랜 정책은 언제든지 변경될 수 있습니다. 현재 제공되는 2주 무료 평가판 등의 조건은 변동될 수 있으니, 최신 정보를 확인하시기 바랍니다.
*   **유료 기능**: 일부 고급 기능이나 특정 규모 이상의 사용량에 대해서는 유료 요금제가 적용될 수 있습니다. 필요에 따라 비용이 발생할 수 있음을 인지해 주십시오.
